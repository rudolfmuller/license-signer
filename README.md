# license

<b>CLI License Generator written in Rust
<br>
POSIX - Only (linux)
</b>
<br>

<b>WARNING:</b> The app isn't perfect, so please help me find any issues (and possible fixes)
<br><br>
<b>HOW TO USE:</b> 
<br>
Add a new license: `$ mv ./MIT.lic ~/.local/share/license-space/`<br>
Generate the license: `$ license gen -t MIT --owner="Rudolf Muller"`.
